# Proje 3: Maç Sonuçları Simülasyonu

Bu proje, Galatasaray, Beşiktaş, Fenerbahçe ve Trabzonspor takımlarının 3 haftalık maçlarını rastgele skorlar üreterek simüle eden bir C# konsol uygulamasıdır.

## Kullanılan Teknolojiler
- C# .NET
- Konsol uygulaması

## Nasıl Çalıştırılır
1. Kod dosyasını bir C# projesine ekleyin (`Program.cs`).
2. Visual Studio veya başka bir IDE ile projeyi açın.
3. Programı çalıştırın ve konsolda maç sonuçlarını ve puan tablosunu görün.

## Özellikler
- Her hafta için 2 maç rastgele skorlarla simüle edilir.
- Puan tablosu ve kazanan takım hesaplanır.
- Konsolda haftalar renkli olarak gösterilir.
